Projects

Energy Management

The application aimed to manage the consumption of 3 kinds of utilities: water, gas and electricity. This project has been developed in two different shapes: as web application and as command line application. As web application has been developed an application according to the MERN stack. Within this I designed the backend, the way in which the information would have been stored into the database and exposed the endpoint to the frontend. The technologies I used are Javascript, MongoDB, NodeJS and Express. The reason why the same application has been developed as CLI app is due to the change in programming paradigm. More specifically this version has been developed as exam project of a course with a focus on Functional Programming. Within this context the application has been developed using the Scala programming language and MongoDB as NoSQL database. During the development of this project I focused on how the information might be represented, used and processed according to the functional paradigm.

CityLife

The project has been realized for the exam of a course with the focus on Smart City paradigm and technologies. This is an Android application with the goal to allow the citizen to signal possible critical situations to all the other citizens that are within a certain notification range specified by the signaler. Along the development of this application I was focused on how the information would have been represented, stored on the database and I focused on the interaction between the frontend (the Android application) and the backend developed by myself. I developed a web service to mediate the access of the frontend to the database. The technology I used in this project have been Kotlin and MongoDB.

SmartParking

The project has been realized for an exam course with the focus on Domain Driven Design and DevOps methodology. The domain of the project is the parking and the application designed and developed aimed to address the issue of finding and reserving a park slot in a proper time. Along this project I was focused on the DevOps pipeline required for the backend, using Kotlin as programming language and GitHub Actions and Docker as technology to implement the pipeline.
